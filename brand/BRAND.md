# Silkway Web Company — бренд-токены

## Идея знака (v3 — актуальная)
Буква S, выложенная из плиток-«пикселей» на изразце лазурита: самаркандская
мозаика = цифровые пиксели, ремесло мастера = код. Два тона золота + один
песочный «включённый пиксель» (свет/курсор) оживляют знак. S = Silkway.
Работает без текста: фавикон, аватар Telegram-бота, вывеска.

Архив v1: лента-дорога с пунктиром. Архив v2: лента на плитке.

## Типографика (план для сайта)
- Display: геометрический гротеск с широким трекингом (в лого: Avenir Next / Futura).
  Для веба подобрать при дизайн-плане (кандидаты: Unbounded, Manrope Display, Golos).
- Body: гуманистический гротеск с поддержкой кириллицы и латиницы (RU/UZ).
- Подпись «WEB COMPANY» всегда золотом, разрядка ~0.7em.
